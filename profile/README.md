# Zona Sur Tech

Construimos software para operar negocios reales en Costa Rica.

Zona Sur Tech desarrolla productos SaaS, ERP, facturación electrónica, automatización con IA e integraciones para PYMEs que necesitan control, velocidad operativa y una base tecnológica seria.

---

## Qué construimos

- ERP y sistema operativo para PYMEs
- Facturación electrónica para Costa Rica
- Automatización con IA y bots
- Integraciones con WhatsApp, APIs y CRMs
- CMS, dashboards y herramientas internas

---

## Enfoque

Diseñamos software con foco en:

- multi-tenant
- seguridad por defecto
- arquitectura modular
- automatización real de procesos
- despliegues listos para producción
- experiencia de usuario directa y funcional
